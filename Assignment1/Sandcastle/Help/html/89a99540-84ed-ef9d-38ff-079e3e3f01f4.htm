      Methods Class

      The class 'Methods' has several private variables such as a boolean noPara_check, a boolean method_found, a boolean endMethod_found, a static int method_index, a static int endmethod_index, a static Dictionary method and a boolean method_call. It also has a private list cmds These variables are used to check for the method calls and handling them. The method_index and endmethod_index are used to keep track of the start and end of the method. The method_found and endMethod_found are used to check if a method and endMethod is found in the code. The noPara_check is used to check if the method has no parameter. The method_call is used to check if a method is called. The method dictionary stores the methods with their name and commands. The cmds list stores the commands of the method.

Inside the while loop it uses the 'Split' method to split the commands on the basis of space. It checks if the first element after splitting is "method" and if the length of split_cmd is 2 and if the second element contains "()". If these conditions are true, it sets the 'noPara_check' variable to true and the 'method_found' variable to true and sets the 'method_index' variable to the current index of the loop. If these conditions are true, it sets the 'endMethod_found' variable to true and the 'endmethod_index' variable to the current index of the loop.
